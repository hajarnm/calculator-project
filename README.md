# Java Calculator Project

## 📌 Project Description
This project is a simple calculator application developed using Java. It allows users to perform basic and advanced mathematical operations through a console-based interface. The project was developed as part of the Software Engineering (SWE 442) course to practice Software Configuration Management (SCM) using Git and GitHub.

---

## ⚙️ Features

### Version 1.0 (Basic Calculator)
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation (e.g., division by zero handling)

### Version 1.1 (Advanced Features)
- Power (a^b)
- Square (a²)
- Square Root (√a)
- Improved menu system
- Better output formatting
